3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:
    Map()- Creats a new array by applying a function to each element.
    filter()- Creates a new array containing only elements that meet a specified condition.
    forEach()- Execute a function for each elements of the array. It doesn't return a new array.
